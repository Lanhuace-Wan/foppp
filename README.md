#### Argument

Program name: Fop

- Path
- `-d`     `--dry_run`
- -`v/-vv` `--verbose`
- `-r`     `--recursive`

  

#### File type

For now, it will take and process these type of file:

- Documents: `txt|docx|doc|pdf|rtf|csv|xls|xlsx|ppt|pptx|odt|ods|odp|key`
- Musics: `mp3|wav|flac|ogg|aiff|wma|m4a`
- Videos: `mp4|avi|mov|wmv|flv|mkv|webm|ogv|mpg|mpeg|m4v`
- Images: `jpg|png|jpeg|gif|bmp|tiff|webp|svg`
- Fonts: `ttf|otf|woff|woff2|eot`
